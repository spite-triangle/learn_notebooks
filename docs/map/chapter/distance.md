# 距离测量

# 概念

距离测量目的
- **水平距离**: 两点连线在水平面上的投影长度
- **倾向距离**：不同水平面上两点间连线长度

测量方案
- **钢尺测量**
- **视距测量**: 精度差
- **电磁波测距**: 昂贵
- **GPS 测距**

## 往返测

为了保证最终测量结果的精度，还会进行两点间距离往返测

$$
D = \frac{1}{2} (|D_1| + |D_2|)
$$

通过相对误差 $K$ 可以衡量往返测量结果的精度

$$
K = \frac{|D_1 - D_2 |}{D}
$$
